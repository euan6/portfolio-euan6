Projects

PHP Backend Project

This was one of my very first big projects which came in my final year of high school. What i created was a review website which used PHP and SQL to query a database to show content of my webpage. There was a login and logout system which functioned correctly and once a user was logged in, only then could they view the contents of the pages. There were a few pages which were interactive including a read, search and add review pages which let the user either view all the reviews in the database, search for a specific one (rating, description, name) or write and submit a review themselves.

Web Programming Project

This web programming project was part of my coursework in my 2nd year at university. The main goal of the project was not to make it fancy but actually to increase my knowledge of git and version control.
